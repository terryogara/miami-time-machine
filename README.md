# 🌴 Miami Time Machine

**A retro-styled terminal narrative adventure set in 1980s South Beach.**

"Miami Time Machine" explores the intersection of classic text-based storytelling and modern Python CLI development, featuring a custom typewriter engine and dynamic visual pacing.

## 🕹️ The Experience

This game places the player in the neon-soaked streets of 1989. Originally developed for a Python coursework assignment, it has since evolved into an experimental project demonstrating how a text-based game transforms from a static read into an immersive experience through timing, ANSI color, and vertical rhythm.

## ⚙️ Technical Features

* **Custom Typewriter Engine:** A simulation engine that adjusts text delivery speed based on punctuation for a natural reading flow.
* **Cross-Platform Fidelity:** Built with [`colorama`](https://www.google.com/search?q=%5Bhttps://pypi.org/project/colorama/%5D(https://pypi.org/project/colorama/)) to ensure consistent ANSI color across Windows, macOS, and Linux.
* **State Management:** A robust input validation system to manage complex narrative transitions.
* **Animated Sequences:** Features custom-built scrolling scramble effects for data decryption and system alerts.

## 🚀 Quick Start

### Prerequisites

* **Python 3.6+**
* `pip` (Python package installer)

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/](https://github.com/)[Your-Username]/miami-time-machine.git
   cd miami-time-machine

```


2. **Install dependencies:**
```bash
pip install -r requirements.txt

```


3. **Run the game:**
```bash
python main.py

```
